# hubot-gocd

hubot script to control gocd

See [`src/gocd.coffee`](src/gocd.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-gocd --save`

Then add **hubot-gocd** to your `external-scripts.json`:

```json
["hubot-gocd"]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
